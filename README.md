# Effects-of-Mental-Imagery-Ability-and-Aging-on-Mental-Rotation

This project involves the creation, usage, and data collection on mental imagery stimuli.

Stimulus_Creation.ipynb can be used to create stimulus on its own by altering the parameters described by the markdwon text in the file.

mental_rotation_app.py is the actual web application that collects data from participants; mental_rotation_app.py can be used either by (1) terminal command python mental_rotation_app.py or (2) double clicking on "Start Mental Rotations.command"
